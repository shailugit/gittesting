The .container class is for resposive fixed width, use it if you need to add padding on side. containers have 15px left and right padding, with no top or bottom padding. To add extra padding use pt-3=add a top padding of 16px Below are the 5 classes in BS-4
  • .col- (extra small devices - screen width less than 576px)
  • .col-sm- (small devices - screen width equal to or greater than 576px)
  • .col-md- (medium devices - screen width equal to or greater than 768px)
  • .col-lg- (large devices - screen width equal to or greater than 992px)
  • .col-xl- (xlarge devices - screen width equal to or greater than 1200px)

    • The .container-fluid class is for resposive to take full width, use if if you need to removed padding on side. The .container-fluid class provides a full width container, spanning the entire width of the viewport
