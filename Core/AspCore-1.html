ASP.NET core is a cross platform, high performance, open source application framework. It is used to build internet based web applications.

Features:
  • Cross platform: Means it can run on Windows/Mac/Linux & can be hosted on IIS, Console, Docker, Apache.
  • Testability: Easy to write test cases, TDD approach
  • Open source: Its a open source
  • Dependency Injection: DI used to build loosly coupled application
  • Modular: Decoupled, Use of Middleware and
  • Model for MVC/WebApi: Both Controller classes under MVC/WebApi inherit from same base class and return IActionResult.

    • Major Components

  • Kestrel: Cross plateform WebServer for ASP.NET core. It can used in combination of reverse proxy server like IIS, Apache

  • Design Patterns:Basic Design Patterns
  • TDD Approach
  • Agile Basics
