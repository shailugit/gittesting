Basic git command

  • command: git init details: initialize the git
  • command: git status detail: give you status
  • command: git add . detail: add all file in folder to local git, its a kind of staging area
  • command: git commit detail: commit the files to git repo
  • command: git log --oneline detail: see logs for commits
  • command: git remote add origin "url" detail: add a remote url/git to your git source to keep it online
  • command: git push -u origin master detail: push local to online git repository named master
  • command: git clone -b master "url" detail: download branch name master to your local machine
